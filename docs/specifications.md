# 📘 Spécifications du Journal du Lycée Ozenne

## 🎯 Objectif Principal
Créer un site web moderne, flexible et interactif pour le journal du lycée Ozenne, permettant la publication et la consultation de contenus variés.

## 🌐 Périmètre du Projet
- Site web accessible au grand public
- Gestion de contenus multimédias
- Espace administration sécurisé

## 📦 Types de Contenus
1. Articles textuels
2. Documents PDF
3. Podcasts
4. Sondages (via Google Forms)

## 🔑 Fonctionnalités Clés

### 1. Gestion de Contenu
- Rubriques dynamiques et personnalisables
- Système de tags/mots-clés
- Publication avec un compte admin unique
- Archivage permanent des éditions

### 2. Interactions Utilisateurs
- Compteur de vues par article
- Système de réactions (likes/cœurs)
- Commentaires avec modération admin
- Moteur de recherche pour anciennes éditions

### 3. Design et Expérience
- Thème saisonnier dynamique
- Design responsive
- Navigation flexible
- Intégration du logo Ozenne

### 4. Fonctionnalités Avancées
- Statistiques pour rédacteurs
- Notifications pour nouveaux articles
- Tableau de bord d'insights

## 🛠 Technologies
- Frontend: React (Vite)
- CMS: Sanity
- Base de données: Supabase
- Styling: Tailwind CSS
- Hébergement: À définir

## 🔒 Sécurité et Accessibilité
- Compte admin unique
- Modération des commentaires
- Conformité WCAG
- Design accessible

## 📊 Métriques et Suivi
- Comptage des vues
- Système de likes
- Statistiques de lecture

## 🚀 Évolutions Futures
- Système de brouillons
- Traduction automatique
- Personnalisation de l'expérience lecteur
